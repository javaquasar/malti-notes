Can, Must, Want and Need

This page brings together the most useful modal-style patterns for daily Maltese: what you can do, what you want, what you must do, and what you need.

nista' irrid nixtieq għandi bżonn

Core idea

  • nista' = can
  • irrid = want / must
  • nixtieq = would like
  • għandi bżonn = need

Core

Core Meaning

What these patterns do

Nista' mmur.I can go.
Irrid naħdem.I want / must work.
Nixtieq nixrob kafè.I would like to drink coffee.
Għandi bżonn nistrieħ.I need to rest.

Useful note

These are not all one single verb type in the same grammatical sense, but for learners they work together as a practical group for everyday speaking.

Ability

Can - nista'

All Pronouns

Jien nista' mmur.I can go.
Int tista' tmur.You can go.
Huwa jista' jmur.He can go.
Hija tista' tmur.She can go.
Aħna nistgħu mmorru.We can go.
Intom tistgħu tmorru.You all can go.
Huma jistgħu jmorru.They can go.

More Models

Nista' ngħin.I can help.
Tista' titkellem bil-Malti?Can you speak Maltese?
Jista' jiġi wara x-xogħol.He can come after work.
Nistgħu nibdew issa.We can start now.
Skill

Know How - jaf

All Pronouns

Jien naf insuq.I know how to drive.
Int taf issajjar.You know how to cook.
Huwa jaf jitkellem bl-Ingliż.He knows how to speak English.
Hija taf tgħum.She knows how to swim.
Aħna nafu naħdmu flimkien.We know how to work together.
Intom tafu taqraw bil-Malti.You all know how to read in Maltese.
Huma jafu jsibu t-triq.They know how to find the way.

When it is useful

Nista' ngħin.I can help.
Naf insajjar.I know how to cook.
Tista' tgħinni?Can you help me?
Taf issib il-bank?Do you know how to find the bank?
Obligation

Want / Must - irrid

All Pronouns

Jien irrid immur.I want / must go.
Int trid tmur.You want / must go.
Huwa jrid imur.He wants / must go.
Hija trid tmur.She wants / must go.
Aħna rridu mmorru.We want / must go.
Intom tridu tmorru.You all want / must go.
Huma jridu jmorru.They want / must go.

More Models

Irrid naħdem illum.I must work today.
Trid tistrieħ.You need / must rest.
Irridu nitilgħu kmieni.We must leave early.
Jridu jaslu fil-ħin.They must arrive on time.
Polite

Would Like - nixtieq

All Pronouns

Jien nixtieq niekol.I would like to eat.
Int tixtieq tiekol.You would like to eat.
Huwa jixtieq jiekol.He would like to eat.
Hija tixtieq tiekol.She would like to eat.
Aħna nixtiequ nieklu.We would like to eat.
Intom tixtiequ tieklu.You all would like to eat.
Huma jixtiequ jieklu.They would like to eat.

More Models

Nixtieq nixrob ilma.I would like to drink water.
Tixtieq tiġi magħna?Would you like to come with us?
Jixtieq jixtri karozza.He would like to buy a car.
Nixtiequ niltaqgħu għada.We would like to meet tomorrow.
Need

Need - għandi bżonn

All Pronouns

Jien għandi bżonn nistrieħ.I need to rest.
Int għandek bżonn tistrieħ.You need to rest.
Huwa għandu bżonn jistrieħ.He needs to rest.
Hija għandha bżonn tistrieħ.She needs to rest.
Aħna għandna bżonn nistrieħu.We need to rest.
Intom għandkom bżonn tistrieħu.You all need to rest.
Huma għandhom bżonn jistrieħu.They need to rest.

More Models

Għandi bżonn tabib.I need a doctor.
Għandek bżonn ftit ħin.You need a little time.
Għandna bżonn nixtru ikel.We need to buy food.
Għandhom bżonn għajnuna.They need help.
Advice

Should - missu

All Pronouns

Jien imissni nistrieħ.I should rest.
Int imissek tmur id-dar.You should go home.
Huwa missu jiekol aktar.He should eat more.
Hija missha torqod kmieni.She should sleep early.
Aħna jmissna nibdew issa.We should start now.
Intom imisskom titilqu kmieni.You all should leave early.
Huma misshom jitkellmu mal-għalliem.They should speak to the teacher.

Advice Models

Imissek tixrob aktar ilma.You should drink more water.
Missu jmur għand it-tabib.He should go to the doctor.
Missha tieħu pawża.She should take a break.
Jmissna nistudjaw aktar.We should study more.
Necessary

It Is Necessary - jeħtieġ

Main Pattern

Jeħtieġ li nitilqu issa.It is necessary that we leave now.
Jeħtieġ aktar ħin.More time is needed.
Jeħtieġ li tkun kwiet.It is necessary that you stay calm.
Jeħtieġ li naslu kmieni.It is necessary that we arrive early.

With Different Subjects

Jeħtieġli nistrieħ.I need to rest / it is necessary for me to rest.
Jeħtieġlek taħdem.You need to work.
Jeħtieġlu jmur il-bank.He needs to go to the bank.
Jeħtieġilha tistrieħ.She needs to rest.
Jeħtieġilna nixtru ikel.We need to buy food.
Jeħtieġilkom titkellmu bil-mod.You all need to speak slowly.
Jeħtieġilhom jaslu fil-ħin.They need to arrive on time.
Negative

Negative Forms

Most useful negatives

Ma nistax immur.I cannot go.
Ma nistgħux nibqgħu.We cannot stay.
Ma nixtieqx noħroġ illum.I would not like to go out today.
M'għandix bżonn karozza.I do not need a car.
Ma nafx insuq.I do not know how to drive.
Ma jmissnix nitkellem hekk.I should not speak like that.

Useful warning

For nista' the negative is a real built form: ma nistax. With għandi bżonn, the negative usually goes on għandi: m'għandix bżonn. With jaf you often use ma nafx / ma tafx / ma jafx. With missu the form often appears as ma jmissnix for "I should not".

Daily Life

Daily Life Models

Illum nista' noħroġ kmieni.Today I can go out early.
Illum irrid naħdem id-dar.Today I must work at home.
Għandi bżonn nixtri ftit ikel.I need to buy some food.
Nixtieq nara lill-ħbieb tiegħi.I would like to see my friends.
Ma nistax noħroġ illejla għax għandi x-xogħol.I cannot go out tonight because I have work.
Nixtieq nistrieħ ftit wara x-xogħol.I would like to rest a little after work.
Q&A

Questions and Answers

Questions

Tista' tiġi illum?Can you come today?
Trid taħdem għada?Do you have to work tomorrow?
X'għandek bżonn?What do you need?
X'tixtieq tiekol?What would you like to eat?

Answers

Iva, nista' niġi.Yes, I can come.
Le, irrid naħdem.No, I must work.
Għandi bżonn ftit ħin.I need a little time.
Nixtieq niekol pizza.I would like to eat pizza.
Dialogues

Mini-dialogues

Meeting a friend

A: Tista' tiġi filgħaxija?A: Can you come in the evening?
B: Le, illum irrid naħdem.B: No, today I must work.
B: Imma għada nista' niġi.B: But tomorrow I can come.

Need and want

A: X'għandek bżonn?A: What do you need?
B: Għandi bżonn mistrieħ.B: I need rest.
B: U nixtieq nixrob kafè.B: And I would like to drink coffee.
Review

Wrong vs Right

Wrong

Nista' mmurx.Mixed form.
Hija irrid tmur.Wrong person form.
Ma għandi bżonnix immur.Wrong negative shape.

Right

Ma nistax immur.I cannot go.
Hija trid tmur.She wants / must go.
M'għandix bżonn immur.I do not need to go.
